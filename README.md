# Unity FPS Prototype — BSc Thesis Project

A first-person shooter prototype developed in Unity 6 with C# for my
BSc thesis at Eszterházy Károly Catholic University.

## Downloads

Download the SourceCode,playable Build and Demo video in release.

## Repository Structure

- `Assets/Scripts/` — Custom gameplay scripts (~29 C# files)
- `Assets/Scenes/` — MainMenu and SampleScene
- `Assets/Tests/EditMode/` — 71 NUnit tests covering core game logic

## How to Run

### Option 1: Play the Build
Download the build ZIP from the link above, extract, and run `FPS_demo.exe`.

### Option 2: Open in Unity
1. Clone this repository
2. Open the project folder in Unity 6 (6000.2.5f1 or later)
3. Open `Assets/Scenes/MainMenu.unity`
4. Press Play

## Controls

- **WASD** — Move
- **W + Shift** — Sprint
- **C** — Crouch
- **Mouse** — Look
- **Left Mouse Button** — Fire
- **E** — Interact
- **1/2** - Switch weapon
- **ESC** — Pause menu
